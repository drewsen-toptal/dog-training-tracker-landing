# Claude Code Restoration Checklist

## Pre-Restoration
[ ] Extract archive: tar -xzf claude-code-backup-20251228.tar.gz
[ ] Verify checksum: md5 (should be 59132c10659a07a58be525c9c0b099a3)
[ ] Read QUICK_START.md or SETUP_INSTRUCTIONS.md
[ ] Ensure Node.js v18+ installed: node --version
[ ] Ensure Python 3.8+ installed: python3 --version

## Configuration Files
[ ] Backup existing config (if any):
    cp ~/Library/Application\ Support/Claude/claude_desktop_config.json \
       ~/Library/Application\ Support/Claude/claude_desktop_config.json.old
[ ] Copy Claude Desktop config:
    cp claude_desktop_config.json ~/Library/Application\ Support/Claude/
[ ] Copy custom MCP servers:
    cp -r custom-mcp-servers/mcp-servers/* ~/mcp-servers/
[ ] Copy OptionsFlow:
    cp -r custom-mcp-servers/optionsflow ~/mcp-optionsflow/

## Python Setup
[ ] Create OptionsFlow venv:
    cd ~/mcp-optionsflow && python3 -m venv .venv
[ ] Activate venv:
    source .venv/bin/activate
[ ] Install dependencies:
    pip install -r requirements.txt
[ ] Deactivate venv:
    deactivate
